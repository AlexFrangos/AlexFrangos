Applied Software Development

I worked for an applied software developer at London’s Great Ormond Street Hospital, where I was the leader of a three-member team. Our project focused on helping to predict unsuccessful surgeries and warn medical staff about them, utilising a network of sensors that produce timestamped data in different rooms of a hospital, for analysing them on the cloud and generating alerts in case of an abnormal situation. The front-end of this was developed using ReactJS and the back-end using Django, while communication between the front-end and the back-end is done via a REST API, with the entire platform hosted on a Microsoft Azure Virtual Machine.
